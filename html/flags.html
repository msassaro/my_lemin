FLAGS

basics

This flag will display the basic recuperation informations. Such as number of ants, start and end room, the rooms and tubes.

matrix

This flag will display the cost matrix (or path matrix). It will only works if the roomlen is lower or equal to 50.

print

This flag will display the two above.

roominfo

This flag will displaymore information concerning the rooms, like the weight per room, the degree per room (or the number of possible path per room) and the dispo per room. All of this before searching paths.

man

This is flag print the user manual on the shell.

errmode

This flag is usefull for better error aprehension. In errmode, the lem-in displays more detailed error messages.

noorder

In normal mode, you have to define all the rooms before the tubes. With this mode, you don't care about the order.

bestpath

This flag will display the best path to go for.

color

This flag will put the ants in different color to easily differentiate them at the output.

redefine_startend{start}{end}

This flag will redefine a new start and a new end. You need to have start and end defined as room that is defined. Same requirements as usual rooms.

redefine_bestpath{start}{end}

This flag will redefine start and end and display the best path between the 2.
