USAGE

./lem-in ( < [valid file] ) ( optional flags )
./lem-in
--> writing directly from STDIN

FILE FORMAT

General format:
1. first line = number of ants. (from 1 to 10 000)
2. you have the rooms definitions
3. you have the tube definitions

You can define the starting room with: ##start
You can define the ending room with: ##end

A room is defined by a name and coordinates.
name x y
with x and y (between 0 and MAX_INT)

The maximum room len is 2000.
The maximum line lenght is 100.

No room with the same name or coordinates. Or error.
